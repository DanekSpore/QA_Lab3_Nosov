◆ TotalExpectedHotelIncomeForTheSeason()

float QA_Lab3_Nosov.Hotel.TotalExpectedHotelIncomeForTheSeason ( )

Метод посчёта ожидаемого дохода отеля за весь сезон.

Возвращает
Метод использует поля класса Room и его метод GetPrice(), вследствие чего возвращает ожидаемый доход отеля, что высчитывается по формуле: $$Income Expected = (Price_{1} + Price_{2}) * Season Duration$$
