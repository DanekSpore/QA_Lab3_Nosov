◆ Attraction()

float QA_Lab3_Nosov.Room.Attraction ( )

Метод посчёта привлекательности номера.

Возвращает
Метод использует поля класса Price и Square и высчитывает привлекательность номера по типу Цена/Площадь. Получается цена за 1 м² - чем больше число, тем дороже. Данное отношение высчитывается по формуле: $$Attraction = Price * Square$$
