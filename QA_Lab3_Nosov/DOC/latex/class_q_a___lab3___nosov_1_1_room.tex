Метод посчёта привлекательности номера. 

\begin{DoxyReturn}{Возвращает}
Метод использует поля класса Price и Square и высчитывает привлекательность номера по типу Цена/Площадь. Получается цена за 1 м² -\/ чем больше число, тем дороже. Данное отношение высчитывается по формуле\+: \$\$\+Attraction = Price $\ast$ Square\$\$ 
\end{DoxyReturn}
